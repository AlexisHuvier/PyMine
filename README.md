# PyMine
 Minecraft Server in Python
 
 WARNING : This server has been tested in 1.14 and could not work on older versions
 
## Dependances

- Python 3
- Quarry (pip install quarry)

## Installation

- Install requirements with `pip install -r requirements.txt`
- Launch PyMine.py
- Enjoy :D

## Features

- Player's connection 
- Chat
